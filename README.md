# GunsmithBot
Discord bot to retrieve weapon rolls for Destiny 2

## Requirements

- [discord.py](https://github.com/Rapptz/discord.py)
- [pydest](https://github.com/jgayfer/pydest)

## Setup

Set your Discord API key to `DISCORD_KEY` as an environment variable
